CKEditor – Prezentare generală

 Top  Previous  Next

CKEditor este un editor online de text, care pot fi încorporate în interiorul pagini web. Este un editor  WYSIWYG (What You See Is What You Get – Ceea ce plasezi este ceea ce vezi), ceea ce înseamnă că textul editat va fi similar cu rezultatul final, după ce documentul este publicat. El implementează în Web unele funcționalități ca și procesoare de text desktop, cum ar fi Microsoft Word sau OpenOffice.

Una dintre cele mai puternice caracteristici de CKEditor este compatibilitatea sa aproape nelimitată. Este o aplicație JavaScript, așa că pur și simplu funcționează cu toate tehnologiile server, la fel ca un element de bază textarea. În ceea ce privește partea de client, CKEditor a fost dezvoltat pentru a fi compatibil cu toate browserele care domină piața și anume Internet Explorer, Mozilla Firefox, Google Chrome, Safari, Opera, etc.

În figura de mai jos este arătat aspectul general al editorului (în CMS este integrat în jQuery dialog - fereastră individuală).
